User List Component

The designer gave us the markup for a new page here: design.html

Create a dynamic version of this page, using lit-element.

  • Clicking on a star should toggle between filled/unfilled.
  • Clicking the button should add a new row with the value from the text field.

See the LitElement Guide for an introduction. Also feel free to use MDL Lite like the designer did. We have included it alrady in the index.html

For your information, modifying an array doesn't cause LitElement to re-render. You can use `this.data = newArray` instead of `this.data.push(...)` to cause a re-render.
